# Tablas aleatorias y Graficos 3d 

Este proyecto genera datos aleatorios según los datos introducidos en las barras de los menus laterales. Una vez configurada la tabla, se generan gráficos en 2D o 3D, según la preferencia del usuario


## Tecnologías

- Python
- Streamlit
- Pandas
- Numpy
- Matplot
- Plotly

## Estructura del proyecto

datos_graficas_2
|-- tabla_graficas_2D_3D.py
|-- README.md


## Cómo ejecutar

1. Clona el repositorio
    git clone https://github.com/fquinzan-GZ/datos_graficas_2.git

2. Entra en la carpeta
   cd mi_dashboard_Streamlit_2

3. Instala las dependencias:
   pip install -r requirements.txt

4. Ejecuta la app:
   streamlit run tabla_graficas_2D_3D.py
